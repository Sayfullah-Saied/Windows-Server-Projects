<h1>SkyBridge Solutions – IT Infrastructure Overview</h1>
SkyBridge Solutions is a mid‑sized IT consulting and services company specializing in cloud integration, infrastructure support, and digital transformation. With a workforce of 50 employees across multiple departments.<br>
As the System Administrator, you are responsible for designing, deploying, and maintaining the core IT infrastructure. Your role includes managing identity services, network configuration, server deployment, security policies, and ensuring high availability of all critical systems across the organization.

## 2. Core Server Infrastructure

| # | Component | Key Functions |
|---|-----------|----------------|
| **1** | **Active Directory Domain Controller** | - Centralized authentication<br>- DNS integrated with AD<br>- Manages users, computers, and Group Policies |
| **2** | **DHCP Server** | - Assigns IP addresses automatically<br>- Separate scopes for each VLAN<br>- Supports network segmentation |
| **3** | **File Server** | - Department shared folders<br>- NTFS permissions based on security groups<br>- Centralized document storage |
| **4** | **Group Policy Management** | - Department‑specific GPOs<br>- Controls security settings, drive mappings, USB policies<br>- Ensures consistent workstation configuration |
| **5** | **Backup System** | - Daily incremental backups<br>- Weekly full backups<br>- Protects AD, file shares, and configs |
| **6** | **Firewall / VPN Gateway** | - Secure remote access<br>- Network protection and filtering<br>- VPN for remote employees |
