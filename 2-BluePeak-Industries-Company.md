# BluePeak Industries – IT Infrastructure Overview

BluePeak Industries is a leading company in IT services and IT consulting.  
As the IT Department, our responsibility is to design, configure, and maintain the technical infrastructure that supports all business units.

### Core Servers Deployed
- **Active Directory Domain Controller**  
  Provides centralized authentication, DNS, and Group Policy management.

- **DHCP Server**  
  Configured with dedicated IP pools for each department, aligned with their VLANs for proper network segmentation.

- **File Server**  
  Hosts departmental shared folders with NTFS permissions based on security groups and roles.

- **Group Policy Management**   
Department‑specific GPOs control software access, security settings, USB policies, drive mappings, and admin rights.

<b>This configuration ensures a secure, scalable, and well‑organized infrastructure that supports all business operations efficiently.</b><hr>
